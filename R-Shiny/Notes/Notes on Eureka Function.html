General Notes and Documentation for Eureka¶

This notebook provides general notes on the Eureka Product and the functions and algorithms that run behing Eureka. The three core functionalities Eureka provides are:

1. Sample Size Estimation¶

Eureka provides automated Sample Size Estimation that inform test design and guide test execution to get a clean and accurate read on the data. The sample size estimation covers both individual sample size, group and variable split sample sizes. Contained in this notebook are the formulars used to estimate the samples

2. Test Segmentation¶

At the core of test design is how do we split groups within test and control such that we are able to get comparable groups for every treatment. Test segmentation will provide a platform for customized and client-agnostic splitting and test segmentation based on input metrics provided to the algorithm.

3. Statistical Testing¶

Perhaps the most important part of the product, Eureka offers several options for performing the test analysis. From traditional techniques of A|B testing to model based analysis like causal impact, Eureka provides a robust list of testing techniques to analyze the results of the data.

1.Sample Size Estimation - Proportion Variable (Binary Outcomes)¶

In the case of sample size estimation, the formula used to calculate the sample size is:

$$ n = \frac {r + 1}{1} \frac {\bar p (1 - \bar p) (Z_\beta + Z_{\alpha/2})^2}{(p_1 - p_2)^2} $$

Where:

$n:$ size of the smaller group
$r:$ ratio of larger group to smaller group
$\bar p :$ Sum of $p_1$ sum lift
$p_1 - p_2:$ Effect size to be measured
$Z_\beta :$ Z critival value for the power level i.e. .84 = 80%
$Z_{\alpha/2}:$ Z Critical value for two-tailed significance level i.e. 1.96 @ $\alpha = .05$

1.1. Simple Calculation of the Sample Size¶

Below, I provide a simple function that will calculate the the sample size given the components provided to the function corresponding to the formular above. Notice that we will need to use the qnorm function to return the critical values associated with Power and Significance level.

from scipy.stats import norm

def sample_calculation(ratio, p_1, lift, power, sig_level):
    """ Computing the Sample by Group """
    
    new_ratio = round(ratio*10/(1-ratio)/10, 2)
    
    z_power = norm.ppf(power)
    z_siglevel = abs(norm.ppf(sig_level/2))
    p_bar = p_1 + lift   
    
    part_1 = (new_ratio + 1)/new_ratio
    part_2 = ((p_bar)*(1 - p_bar)*(z_power + z_siglevel)**2)/(lift)**2
    sample = part_1*part_2
    
    return int(round(sample))

1.2. Sample Size Estimation - Countinuous Variable Outcome¶

In the above example, we have seen the estimation of sample size from a binary outcome. We can also do the same for the continuous variable outcome. The general formular we will use here is the following:

$$ n = \frac {r + 1}{r} \frac {\sigma^2 (Z_{beta} + Z_{\alpha/2})^2 } { difference^2 } $$

Where:

$n:$ size of the smaller group
$r:$ ratio of larger group to smaller group
$\sigma :$ standard-deviation of the characteristic
$difference:$ differences in means of the outcome
$Z_\beta :$ Z critival value for the power level i.e. .84 = 80%
$Z_{\alpha/2}:$ Z Critical value for two-tailed significance level i.e. 1.96 @ $\alpha = .05$

We can use the above formula for estimating univariate sample needed for detecting lift.

In the example below, we develop the code for providing this sample estimation.

def ContinuousSampleEstimation(variable_series, mean_diffence, power, significance, ratio):
    
    """ This function will compute the sample Estimates for Continuours Variables"""
    z_power = norm.ppf(power)
    z_alpha = abs(norm.ppf(significance/2))
    variance = np.var(variable_series)
    
    ratio_part = (ratio + 1)/ratio
    sample_part = variance*(z_power + z_alpha)**2/(mean_diffence**2)
    
    return math.ceil(ratio_part*sample_part)

1.3 Kolmogorov Smirnov Power Analysis - (Non-normal Continuous Data)¶

Most of our data may not be normally distribution or we may wish to test for continuous data points for non-normally distribution data. In this case, we would need to run a Kolmogorov Smirnov Simulation to determine what sample size required at every specified time frame.

Below is the proposal and implementation of this.

  1. Given some time series A (and say B), we can simulate what the time series would be in the next 4- six weeks
  2. With every prediction/ we simulate and lift based on a random normal distribution with lift as the mean
  3. Perform a Kolmogorov Smirnov Test after the simulations to determine the lift if impact
  4. Look at the proportion of simulations the determined significance and convert to power proportion
